clear (清除) 屬性是用來抵銷 float 屬性的作用。可能的值為：

  • left：消除左邊的浮動。
  • right：消除右邊的浮動。
  • both：消除左邊及右邊的浮動。
  • none：不消除任何一邊的浮動。

來看一下從 float 那一頁稍作修改的例子：

CSS 宣告：

#leftfloat {

float:left;

}


#clearleft {

clear:left;

}

銷 float 屬性的值。</span>

這個例子展現出 clear:left 能夠抵銷 float 屬性的值。

clear:left樣式後的所有文字都是在圖案之下。
