CSS 教學 > 浮動


一個在大型網站常見的格式，就是將文字繞著一個圖案顯現。這是用 float (浮動) 這個屬性來達成

的。

float 屬性有三個可能的值：left、right、和 none。來看看以下的例子：


例 1：若有以下的 CSS 宣告，

#leftfloat {

float:left;

}

那以下的 HTML 碼，

<span id="leftfloat"><img src="yp.jpg"></span>這個例子是用來顯示 float:left 會如何影響整個頁面

的呈現。在這裡，圖案浮動於左邊。

就會顯現為，

這個例子是用來顯示 float:left 會如何影響整個頁面的呈現。在這裡，圖案浮動於左邊。

例 2：若有以下的 CSS 宣告，

#rightfloat {

float:right;

}

那以下的 HTML 碼，

<span id="rightfloat"><img src="yp.jpg"></span>這個例子是用來顯示 float:right 會如何影響整個頁面

的呈現。在這裡，圖案浮動於右邊。

就會顯現為，

這個例子是用來顯示 float:right 會如何影響整個頁面的呈現。在這裡，圖案浮動於右邊。
